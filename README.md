# A little bit about Linux Commands

 $ ls: lista arquivos
 $ ls -a: lista arquivos ocultos
 $ pwd: mostra o diretório atual
 $ cat: mostra o conteudo de um arquivo

 - Sair da edição: ESC ou [i]
 - Salvar: [w]
